   - 復元：「右回転を行う関数rotateRのプログラムを図5に、左回転を行う関数rotateLのプログラムを図6に示す。これらの関数は、回転した結果として得られた木の根への参照を返す。」
     ```
     // t が参照するノードを根とする木に対して右回転を行う
     function rotateR(t)

     endfunction

     ```
