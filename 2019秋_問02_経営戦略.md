# 2019年秋期（令和元年度）応用情報技術者試験 午後 問2（選択）
## 経営戦略：スマートフォン製造・販売会社の成長戦略（B社）

---

## 問題文

**問2** スマートフォン製造・販売会社の成長戦略に関する次の記述を読んで、設問1〜4に答えよ。

B社は、スマートフォンの企画、開発、製造、販売を手掛ける会社である。"技術で人々の生活をより豊かに"の企業理念の下、"ユビキタス社会の実現に向けて、社会になくてはならない会社となる"というビジョンを掲げている。これまでは、スマートフォン市場の拡大に支えられ、順調に売上・利益を成長させてきたが、今後は市場の拡大の鈍化に伴い、これまでのような成長が難しくなると予測している。そこで、B社の経営陣は今後の成長戦略を検討するよう経営企画部に指示し、同部のC課長が成長戦略検討の責任者に任命された。

---

### 〔環境分析〕

C課長は、最初にB社の外部環境及び内部環境を分析し、その結果を次のとおりにまとめた。

**(1) 外部環境**

- 国内のスマートフォン市場は成熟してきた。一方、海外のスマートフォン市場は、国内ほど成熟しておらず、伸びは鈍化傾向にあるものの、今後も拡大は続く見込みである。日本から海外への販売機会がある。
- 国内では、国内の競合企業に加えて海外企業の参入が増えており、競争はますます激しさを増している。これによって、多くの企業が市場を奪い合う形となり、価格も下がり`[　a　]`となりつつある。
- 5Gによる通信、IoT、AIのような技術革新が進んでおり、これらの技術を活用したスマートフォンに代わる腕時計のようなウェアラブル端末や、家電とつながるスマートスピーカの普及が期待される。また、医療や自動運転の分野で、新しい機器の開発が期待される。一方で、技術革新は急速であり、製品の陳腐化が早く、市場への迅速な製品の提供が必要である。
- スマートフォンは、機能の豊富さから若齢者層には受け入れられやすい。一方で、操作の複雑さから高齢者層は使用することに抵抗があり、普及率は低い。
- スマートフォンへの顧客ニーズは多様化しており、サービス提供のあり方も重要になっている。

**(2) 内部環境**

- B社は自社の強みを製品の企画、開発、製造の一貫体制であると認識している。これによって、顧客ニーズを満たす高い品質の製品を迅速に市場に提供できている。また、単一の企業で製品の企画、開発、製造をまとめて行うことで、異なる製品間における開発資源などの共有を実現し、複数の企業に分かれて企画、開発、製造するよりもコストを抑えている。
- B社は国内の販売に加えて海外でも販売しているが、マニュアルやサポートの多言語の対応などでノウハウが十分でなく、いまだに未開拓の国もある。
- B社はスマートフォンの新機能に敏感な若齢者層をターゲットセグメントとして、テレビコマーシャルなどの広告を行っている。広告は効果が大きく、売上拡大に寄与している。一方で、高齢者層は売上への寄与が少ない。
- B社は医療や自動運転の分野の市場には販売ルートをもっておらず、これらの市場への参入は容易ではない。
- 競合企業の中には製造の体制をもたない、いわゆるファブレスを方針とする企業もあるが、B社はその方針は採っていない。**①今後の新製品についても、現在の方針を維持する予定である。**

---

### 〔成長戦略の検討〕

C課長は、環境分析の結果を基に、ビジネス`[　b　]`の一つである成長マトリクスを図1のとおり作成した。図1では、製品・サービスと市場・顧客を四つの象限に区分した。区分に際しては、スマートフォンを既存の製品・サービスとし、スマートフォン以外の機器を新規の製品・サービスとした。また、現在販売ルートのある市場の若齢者層を既存の市場・顧客とし、それ以外を新規の市場・顧客とした。

### 図1 成長マトリクス

|  | 既存（製品・サービス） | 新規（製品・サービス） |
|---|---|---|
| **既存（市場・顧客）** | 第1象限：・新機能の追加<br>・`[　c　]` | 第2象限：・スマートスピーカの製品化<br>・（省略） |
| **新規（市場・顧客）** | 第3象限：・現在販売ルートのある市場の高齢者層への販売<br>・（省略） | 第4象限：・医療機器の製品化<br>・（省略） |

当初、C課長は、成長マトリクスを基に外部環境に加えて内部環境も考慮して検討した結果、**②第2象限と第4象限の二つの象限の戦略に力を入れるべきだと考えた**。しかし、その後**③第4象限の戦略に関するB社の弱みを考慮し**、第2象限の戦略を優先すべきだと考えた。

---

### 〔投資計画の評価〕

第2象限の一部の戦略については、すぐにB社で製品化できる見込みのものがある。内部環境を考慮すると、これについてもB社で企画、開発、製造を行うことで、`[　d　]`によるメリットが期待できる。

C課長は、この製品化について、複数の投資計画をキャッシュフローを基に評価した。投資額の回収期間を算出する手法としては、金利やリスクを考慮して将来のキャッシュフローを`[　e　]`に割り引いて算出する割引回収期間法が一般的な方法であるが、製品の陳腐化が早いので簡易的な回収期間法を使用することにした。また、回収期間の算出には、損益計算書上の利益に**④減価償却費を加えた金額**を使用した。製品化の投資計画は、表1のとおりである。

### 表1 製品化の投資計画（単位：百万円）

## 解答と解説

### 設問1

**a = エ（レッドオーシャン） / b = イ（フレームワーク） / c = ウ（提供サービスの細分化） / d = エ（範囲の経済）**

- a：競合が激化し価格競争に陥った既存市場は「血みどろの海」＝**レッドオーシャン**と呼ばれる。対義語は競争のない新市場「ブルーオーシャン」。
- b：成長マトリクス（アンゾフの成長マトリクス）は、ビジネス**フレームワーク**（経営分析の型・手法）の一つ。
- c：第1象限は「既存の製品・サービス（スマートフォン）×既存の市場・顧客」。本文の外部環境「顧客ニーズは多様化しており、サービス提供のあり方も重要」に対応する施策は**提供サービスの細分化**。
- d：単一企業が複数の関連製品を企画・開発・製造することで得られるコスト・シナジーのメリットは**範囲の経済**（Economies of Scope）。「規模の経済」は同一製品の生産量拡大によるコスト低減であり、本文の「異なる製品間における開発資源などの共有」というシナジーの説明とは区別される。

**IPA公式：a = エ、b = イ、c = ウ、d = エ**

---

### 設問2

**正解：エ（高い品質の製品をコストを抑えて製造したい。）**

下線①「今後の新製品についても、現在の方針（企画・開発・製造の一貫体制、非ファブレス）を維持する」の目的は、内部環境の記述「顧客ニーズを満たす高い品質の製品を迅速に市場に提供できている」「複数の企業に分かれて企画、開発、製造するよりもコストを抑えている」に対応する。すなわち**高い品質の製品をコストを抑えて製造したい**という狙い。

**IPA公式：エ**

---

### 設問3

**(1) 正解（40字以内）：企画から製造の一貫体制を強みに、低コストで高品質の製品にできるから**

第2象限（スマートスピーカ等）・第4象限（医療機器等）はいずれも新規の製品・サービスであり、B社の内部環境の強みである「企画、開発、製造の一貫体制」を生かせば、他社より低コストで高品質な新製品を提供できるという積極的な理由がある。

**IPA公式：企画から製造の一貫体制を強みに、低コストで高品質の製品にできるから**

**(2) 正解（25字以内）：医療や自動運転の市場には販売ルートがないこと**

内部環境の記述に「B社は医療や自動運転の分野の市場には販売ルートをもっておらず、これらの市場への参入は容易ではない」とあり、第4象限（医療機器の製品化等）を進める上でのB社の弱みは**医療や自動運転の市場には販売ルートがないこと**。

**IPA公式：医療や自動運転の市場には販売ルートがないこと**

---

### 設問4

**(1) e = 現在価値（3字）**

割引回収期間法は、将来のキャッシュフローを金利やリスクを考慮した割引率で**現在価値**に割り引いて評価する手法。

**IPA公式：現在価値**

**(2) 正解（30字以内）：減価償却費はキャッシュの移動がない費用だから**

損益計算書上の利益は発生主義に基づき減価償却費を費用として差し引いた後の金額だが、減価償却費は実際に現金が社外に流出しない非資金費用である。回収期間の算出はキャッシュフローベースで行うため、利益に減価償却費を足し戻す必要がある。理由は**減価償却費はキャッシュの移動がない費用だから**。

**IPA公式：減価償却費はキャッシュの移動がない費用だから**

**(3) f = 2.2（年）**

年間キャッシュフロー（利益＋減価償却費）は、1年目：200+200=400、2年目：300+200=500、3年目：300+200=500、4年目：200+200=400、5年目：100+200=300百万円。

投資額1,000百万円の累計回収を追うと、1年目末に400（累計400）、2年目末に500（累計900）、3年目中に残り100を回収するのに 100÷500年＝0.2年 かかる。したがって回収期間は 2年＋0.2年＝**2.2年**。

**IPA公式：2.2**

## 参考：主要キーワード

| 用語 | 説明 |
|------|------|
| 成長マトリクス（アンゾフのマトリクス） | 製品・サービスと市場・顧客の既存／新規の組合せで事業成長戦略を4象限に整理するフレームワーク |
| レッドオーシャン／ブルーオーシャン | 競争が激しい既存市場と、競争のない未開拓の新市場を対比する概念 |
| 範囲の経済（Economies of Scope） | 複数の異なる製品・事業を一つの企業が扱うことで得られるコスト削減効果（資源の共有） |
| 規模の経済（Economies of Scale） | 生産量の拡大によって単位当たりコストが低減する効果 |
| ファブレス | 自社で製造設備をもたず、企画・開発に特化し、製造は外部委託する事業方針 |
| 割引回収期間法／現在価値 | 将来のキャッシュフローを金利・リスクを考慮した割引率で現在価値に換算し、投資回収期間を評価する手法 |
| 減価償却費と非資金費用 | 減価償却費は会計上の費用だが実際の現金支出を伴わないため、キャッシュフロー計算では利益に加算し戻す |
