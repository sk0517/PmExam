# 2023年秋期（令和5年度秋期）応用情報技術者試験 午後 問2（選択）
## 経営戦略：バランススコアカード・SECIモデルによるソリューション営業戦略

---

## 問題文

**問2** バランススコアカードを用いたビジネス戦略策定に関する次の記述を読んで、設問に答えよ。

X社は、大手の事務機器販売会社である。複写機をはじめ、様々な事務機器を顧客に提供してきた。顧客の事業環境の変化の中の収益の低下によって、X社の利益率は低下傾向にある。そこで、X社経営部は、数年前に複数のIT関連の商品やサービスをビジネスとするソリューションビジネスに着手した。

X社のソリューションビジネスの市場認知度を高める必要があるが、現状では、顧客に訴求できるような情報発信力が不足している。

提案活動の参考にする過去のソリューション事例を、サーバに登録することにしている。しかし、登録数は多いが自身の経験を公開することへの不安から人事評価につながらないため登録数は積極的でなく、現在は蓄積されている件数が少ない。また、有益な情報があっても探すのに時間がかかり、提案のタイミングを逃して失注している。

### 〔ビジネス戦略の施策〕

現状分析を踏まえて、Y室長はビジネス戦略の施策を次のようにまとめ、これらの施策を実施することにした。ROE を 10%以上に伸ばさせることとした。

**(1) 人材開発**
- ソリューション人材を育成する仕組みを確立する。具体的には、ソリューション人材の営業ノウハウを知識化して社内に共有することとし、ソリューション提案の研修を開催し、ソリューションの知識や顧客のニーズを掘り出すスキル、提案が得られないような営業の発信力の向上に利活用する。
- 人事評価制度を見直し、営業担当者は売上高を目標値に加え、ソリューション事例の参照ごとに組織の営業力を高めることへの貢献度を評価する。また、S企画本部の担当者に対しては、ソリューションごとの顧客満足度と販売実績の利益率を評価する。さらに、人材開発本部が担当する人材には、開催した研修によって育成したソリューション人材の人数を評価する。

**(2) ソリューション開発**
- 顧客の事業環境の変化に対応してソリューションのぞろえを増やすため、専任チームを立ち上げ、多様な商品やサービスをもつサービス事業者との業務提携を拡大する。業務提携によっては、`[　a　]` 制度を、そのサービス事業者から合法に取得することによって他社との差別化を図る。
- ソリューション事例を整備し、顧客のニーズ・課題及び予算に入力することで、該当するソリューションパターン（以下、ソリューションパターンという）を整備する。

**(3) 営業活動**
- ソリューションパターンの提案を増やすことによって、顧客価値と利益率が高いソリューションビジネスの先上拡大を図る。
- X社のソリューションの市場認知度を高めるために、ソリューション事例を顧客に訴求できる魅力的な情報を提供するなど、コンテンツマーケティングを行う。
- 顧客の真のニーズを満たす顧客価値を提供するために、営業本部と S 企画本部とが協力して開発するソリューションを活用して、営業活動を展開する。

## 解答と解説

### 設問1

**(1) 正解：全社レベルで統一されたビジネス戦略を描くこと**

各部門の担当者を集めることで、財務・顧客・業務プロセス・学習と成長の4視点を全社横断的に整合させ、統一されたビジネス戦略を描くことができる。

**(2) 正解：a=独占的に販売できる（10字）**

サービス事業者との業務提携で、そのサービスを「独占的に販売できる」権利（代理店契約・専売契約等）を取得して他社との差別化を図る。

**(3)**
- **b=コンテンツマーケティング**
- **d=業務提携するサービス事業者**

顧客視点のアクションは「コンテンツマーケティング」で市場認知度を向上。業務プロセスの達成目標は「業務提携するサービス事業者（数）」。

**(4) 正解：c=ウ（ソリューションパターンを活用した営業活動）**

業務プロセス視点の評価指標として、提案件数とともに「ソリューションパターンを活用した営業活動」数を測定する。

---

### 設問2

**(1) 正解：C, D, B, A**

SECIモデルの順番：
- **共同化（S）**：C（営業担当者の事例を組織全体で共有→暗黙知の共有）
- **表出化（E）**：D（ノウハウを教材化・形式知化）
- **連結化（C）**：B（ソリューションパターンとして社内展開・形式知の組み合わせ）
- **内面化（I）**：A（実際の営業活動での活用・実践による学習）

**(2) 正解：事例を登録する行動**

スコア付与によって営業担当者は「事例を登録する行動」を促される。これにより事例の蓄積が進む。

**(3) 正解：顧客の真のニーズに合ったソリューションをタイムリーに提案できる。（35字）**

顧客情報を入力するだけで最適なソリューションパターンが提示されるため、適切なタイミングで顧客ニーズに合った提案が可能になる。

---

### 設問3

**正解：e=15**

ROE = 売上高当期純利益率 × 総資産回転率 ÷ 自己資本比率

= 4.0% × 1.5回 ÷ 40% = 6.0% ÷ 0.4 = **15%**

（デュポン分析：ROE = ROA ÷ 自己資本比率 = 6.0 ÷ 0.4 = 15）

## 参考：主要キーワード

| 用語 | 説明 |
|------|------|
| バランススコアカード（BSC） | 財務・顧客・業務プロセス・学習と成長の4視点から戦略を管理するフレームワーク |
| 重要成功要因（CSF） | 戦略目標の達成に欠かせない要因 |
| 評価指標（KPI） | 目標達成度を測定する定量的な指標 |
| SECIモデル | 野中・竹内が提唱した知識創造モデル（共同化→表出化→連結化→内面化） |
| 共同化（Socialization） | 暗黙知を体験・共感を通じて他者に伝達するプロセス |
| 表出化（Externalization） | 暗黙知を言語・図式などの形式知に変換するプロセス |
| 連結化（Combination） | 形式知同士を組み合わせて新たな形式知を生み出すプロセス |
| 内面化（Internalization） | 形式知を実践を通じて暗黙知として体得するプロセス |
| ROE | Return on Equity。自己資本利益率 = 当期純利益 ÷ 自己資本 |
| デュポン分析 | ROE = 売上高利益率 × 総資産回転率 ÷ 自己資本比率 に分解する財務分析手法 |
| コンテンツマーケティング | 有益なコンテンツを提供して顧客を引き付けるマーケティング手法 |
| ソリューションパターン | 顧客課題に対応した解決策のテンプレート |
