# 2022年春期（令和4年度春期）応用情報技術者試験 午後 問2（選択）
## 経営戦略：化粧品製造販売会社でのゲーム理論を用いた事業戦略の検討

---

## 問題文

**問2** 化粧品製造販売会社でのゲーム理論を用いた事業戦略の検討に関する次の記述を読んで、設問1〜3に答えよ。

A社は、国内大手の化粧品製造販売会社である。国内に八つの工場をもち、自社で企画した商品の製造を行っている。販売先は約30の販売子会社と約200の直営店のほか、売上に加盟店契約を結んだ約2万の化粧品販売店（以下、加盟店という）がある。卸売会社を通じずに加盟店を通じた販売チャネルを一本化して、自社製品を販売してきた。加盟店が加盟店料を支払うことで自社製品を独占販売する見返りに、販売奨励のための粗利益率が高い商品をA社の美容販売員の派遣などのA社独自の手厚い支援を通じて、良好な对顧関係を築いてきた。化粧品販売に直接携わる加盟店スタッフが積極的に顧客へのニーズを掘り、A社の事業は加盟店の販売による店頭販売が支えてきた。また、各工場に隣接した物流倉庫から各店舗への配送は自社が担ってきた。

取扱商品は、スキンケアを中心にヘアケア、フレグランスなど、幅広く揃えており、粗利益率の高い中高価格帯の商品が売上全体の70%以上を占めている。

---

### 〔A社の事業の状況と課題〕

A社の昨年度の売上高は7,600億円、営業利益は800億円であった。A社は、戦略的顧客点着に商品質なイメージとブランドの維持に努め、工場及び直営店を自社で保有し、積極的に広告宣伝及び研究開発費に投入している。A社では、売上高にかかわらず一定を設備に係る費用、売上高の生産能力を拡大し、更に新工場を建設する。そのため、総費用に占める固定費の割合が高い状態が営業利益にどのような影響をもたらすかが課題であった。

A社の過去3年の売上高及び営業利益の推移は横ばいだった。A社は、これまで既緯を経営をかしてわ市場シェアを拡大し、売上を増やすことによって影業利益を増やすかという戦略に取り組んできたが、景気の見通しが不透明であることから、景気が悪化しても安定した営業利益を確保することを今後の経営の事業方針とした。①**これまでの事業戦略は今後の経営の事業方針に適合していない**ので、固定費と変動費の割合の観点から費用構造を見直し、これに沿った事業戦略の策定に着手した。

---

### 〔ゲーム理論を用いた事業戦略の検討〕

事業戦略の検討を指示された経営企画部は、まず固定費の中で金額が大きい自社の工場への設備投資に着目し、今後の設備投資に関して次の三つの案を掲げた。

(1) 増機案：全8工場の生産能力を拡大し、更に新工場を建設する。
(2) 現状維持案：全8工場の生産能力を現状維持する。
(3) 消機案：主要6工場の生産能力を現状維持し、主にヘア製品を製造している2工場を閉鎖する。

景気の見通しは不透明で、その予測は難しい。ここで、②**設備投資案から一つの案を選択する場合の意思決定の判断材料の一つとしてゲーム理論を用いることが有効**だったが、A社の事業方針に従い `[　a　]` に基づくと、消機案が最適になることが分かった。

次に、これから最も強力な競合相手となるプレイヤーを加えたゲーム理論を用いて両社の対戦的な戦略を検討した。トイレタリー事業大手B社が、3年前に化粧事業に参入してきた。強力な既存の流通ルートを生かし、現在は低価格帯の商品にドラッグストアやコンビニエンスストアで販売することで、低価格帯での市場シェアを獲得してきている。現在のA社とB社の全体市場シェアは A社が38%、B社が24%である。今後、中高価格帯の商品の市場規模は現状維持して、低価格帯の商品の市場規模が拡大すると予測しているので、両社の全体市場シェアの差は以後に縮まると懸念している。

経営企画部は、これを受けて今後A社が注力すべき商品の価格帯に関して次の二つの案を挙げた。

- A1案（中高価格帯に注力）：粗利益率の高い中高価格帯の商品の割合を増やす。
- A2案（低価格帯に注力）：売上高の割合が増える低価格帯の商品の割合を増やす。

これに対し、B社もB1案（中高価格帯に注力）又はB2案（低価格帯に注力）から一つの案を選択することになる。同社の強みを踏まえる上での、中高価格帯の市場で競合した場合は、A社が有利に中高価格帯の市場シェアを獲得できる。逆に、低価格帯の市場で競合した場合は、B社が有利な可能性がある。

後日、経営企画部は、設備投資案及び注力すべき商品の価格帯の検討を踏まえて事業戦略を策定し、その内容を経営会議に報告した。その内に `[　a　]` に基づいて設備投資の意見が出た、一方で注力すべき商品の価格帯については次の意見が出た。「景気の低迷で価格を中高価格帯の商品に注力することを全面に打ち出すことにすると、中心となる利用者が高齢化により、将来的に中高価格帯市場でも脅かされるのではないか」という意見が出た。

### 〔事業戦略案の策定〕

経営企画部は、前回の経営会議での意見に従って事業戦略案を策定し、再び経営会議で報告した。

(1) 売上高重視から収益性重視への転換
- 低価格帯中心の商品であるヘアケア分野から撤退する。
- 主要6工場の生産能力は現状維持し、主にヘアケア商品を生産している2工場を閉鎖する。
- 不採算の直営店及び直営店を現在の数の約200から半数に縮小させる。

(2) 新たな商品ラインの開発
- 若者層向けのエントリモデルとして低価格帯の商品を拡充する。中高価格帯の商品とは異なるブランドを作り、販売チャネルも変える。具体的には、自社製造ではなく⑤**OEM メーカに製造を委託**して要望の変化に応じて生産する。また、直営店や加盟店ではなく⑥**ドラッグストアやコンビニエンスストアで販売**し、A社の実態販売員の販路が利用しない。

(3) デジタル技術を活用した新たな事業モデルの開発
- インターネットを介した中高価格帯の商品販売などのサービス（以下、ECサービスという）を開発する。2年後のECサービスによる売上高の割合を30%台にすることを目標にする。
- 店舗サービスとECサービスとを連携させて、顧客との接点を増やす顧客総合システムを開発する。
- 新たな事業モデルにおけるECサービスでは、主に顧客が ECサービスを利用して購入したい商品が見つかったときの初めての機会に、顧客が希望する Web 上での充実した情報を相談する。顧客がその加盟店に訪れるのが初めての場合も、美容販売員はチームが ECサービスを利用した際に相談報告した申告から期待するカウンセリングやアドバイスを行うことができる。急激的な商品販売記録が期待できる。⑦**この事業モデルでは店舗サービスとECサービスとが融合できる**ことを関係者に短期間以内に理解してもらう。

経営企画部の事業戦略案は承認され、実行計画の策定に着手することになった。

## 解答と解説

### 設問1

**(1) 正解：イ（正社員の人件費）**

固定費 = 売上高に関係なく一定額が発生する費用。正社員の人件費は売上に関わらず毎月固定で発生する。ア（原材料費）・ウ（外注製造費）・エ（配送外注費）は売上・生産量に応じて変動する変動費。

**IPA公式：イ**

**(2) 正解：売上高の増減に対して営業利益の増減幅が大きくなる。（27字）**

固定費割合が高いと、損益分岐点が高くなり、売上が減少した場合に営業利益が急激に悪化する（営業レバレッジが高い）。景気悪化で売上が落ちると、大きな損失が発生するリスクがある。

---

### 設問2

**(1) 正解：景気の見通しの予測は難しいが営業利益は予測できる。（25字）**

ゲーム理論は「相手の行動が不確実な状況」での意思決定に有効。景気の見通しは予測困難だが、各景気シナリオ下での営業利益は表1のとおり予測できる。このような不確実性がある意思決定に有効。

**(2) 正解：a = オ（マクシマックス原理）、b = イ（ナッシュ均衡）**

- **a = オ（マクシマックス原理）** を使うと消機案が最適：各案の最大値→増機案1,200・現状維持965・消機案800のうち最大を選ぶと増機案が選ばれる。ただし、A社の方針（安定志向）に従い **マクシミン原理**（最悪の場合の最善を選ぶ：消機案）が正解。
  - ※IPA公式：**a = オ（マクシミン原理）**（各案最小値→増機案640・現状720・消機案740→最大の消機案）

- **b = イ（ナッシュ均衡）**：一方が戦略を変えても利益が増えない均衡状態。A1案・B2案の組み合わせでは、どちらも戦略変更の動機がない。

**IPA公式：a = オ（マクシミン原理）、b = イ（ナッシュ均衡）**

**(3) 正解：中高価格帯の商品は粗利益率が高いから。（21字）**

A1案（中高価格帯注力）を選択した場合、市場シェアがA1案でのシェア（37%→41%方向）に留まっても、粗利益率の高い中高価格帯の販売増加により営業利益が増加する可能性がある。

---

### 設問3

**(1) 正解：固定費の割合の減少（9字）**

- OEM委託（⑤）：自社工場での生産を委託することで、工場固定費（設備投資・人件費）を削減し、変動費化できる。
- ドラッグストア・コンビニ（⑥）：自社直営店・加盟店網の維持コスト（固定費）を削減し、流通コストを変動費化できる。
→ 両者の共通点：**固定費の割合が減少する**。

**(2) 正解：顧客は実際に商品を試してから購入したい。（22字）**

化粧品は実際に試してみてから購入する特性（体験・カウンセリングの重要性）があるため、店舗サービス（試用・カウンセリング）とECサービス（事前情報収集・購入）を組み合わせたモデルが成立する。

## 参考：主要キーワード

| 用語 | 説明 |
|------|------|
| ゲーム理論 | 複数のプレイヤーが戦略的に行動する状況の数学的分析手法 |
| マクシミン原理 | 各選択肢の最悪結果の中で最良のものを選ぶ意思決定基準（リスク回避型） |
| マクシマックス原理 | 各選択肢の最良結果の中で最良のものを選ぶ意思決定基準（楽観型） |
| ナッシュ均衡 | どのプレイヤーも単独では戦略を変えることで利益を増やせない均衡状態 |
| 固定費 | 売上高・生産量に関係なく一定額発生する費用（人件費・設備減価償却等） |
| 変動費 | 売上高・生産量に比例して変動する費用（原材料費・外注費等） |
| 営業レバレッジ | 固定費割合が高いほど売上変動が営業利益に大きく影響すること |
| OEM（Original Equipment Manufacturing） | 他社ブランドの製品を受託製造すること。委託側は固定費を削減できる |
| 損益分岐点 | 売上高＝費用（固定費＋変動費）となる売上高。固定費が高いほど高くなる |
